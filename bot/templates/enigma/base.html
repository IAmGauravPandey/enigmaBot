About Enigma



Hey there, I'm Enigma, the most feature-rich Zulip Chat Bot built for solving your queries. I was developed by the Team Enigma Rebooted in the HackFest 2k19, the offcial Fest of Department of Computer Science and Engineering, IIT(ISM) Dhanbad.I'm famous for solving various programming queries and, am really good in mind games like Hangman,Memory game and Scrabble.
So? Up for a challenge? Follow the upcoming steps and Voila!

Features

1. Chatting

2. Description of any Linux Command.

7. Mind Games

Instructions to Use Me

1. Create a Zulip Realm

2. Goto to settings and create a new generic bot named 'Enigma'. (Settings can be found in dropdown of gear icon present in top right corner of zulip realm)

3. Download the zuliprc file for your bot and place it in your home directory as '.zuliprc'.

4. Install all the requirements using pip install -r requirements.txt

5. In bot.py , change site in self.client = zulip.Client(site="https://chunkzz.zulipchat.com/api/") to url of your created zulip realm.Do the same for

BOT_MAIL variable

6. Run bot.py> using the lastest version of Python. python bot.py

7. Head over to your created zulip realm and start using the bot.
